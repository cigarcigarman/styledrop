# PRD — StyleDrop

## 핵심 가설
"패션에 관심 있는 일반인은 자신의 스타일 아이디어를 시각화하고 싶지만, 포토샵이나 스타일리스트에게 의뢰할 도구/예산이 없다."

## 타겟 유저
- 1순위: 패션 인플루언서 지망생, 스타일 블로거 (20-30대 여성)
- 2순위: 의류 소호몰 운영자 (룩북 컨셉 시각화)
- 3순위: 패션 학과 학생

## 핵심 기능 (MVP)
1. 텍스트 → 패션 이미지 생성 (Replicate FLUX)
2. 생성 이력 갤러리 (본인 것만)
3. 크레딧 구매 (Stripe)
4. 이미지 다운로드

## 비-기능 (MVP에서 제외)
- 소셜 기능 (좋아요, 팔글로우)
- 커뮤니티 갤러리
- 모델 파인튜닝
- 모바일 앱
