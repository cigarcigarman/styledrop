# PRD — StyleDrop

## 0. 핵심 정의

| | |
|---|---|
| **문제** | 팬이 좋아하는 작가 화풍으로 이미지를 갖고 싶은데 방법이 없다 |
| **해결** | 작가 동의 기반 LoRA로 즉시 생성 + 다운로드 |
| **기능** | 프롬프트 입력 → 이미지 생성 → 다운로드 |
| **BM** | 생성 1회 = 990원 |

---

## 1. 수익 구조

### 건당 실제 수익
```
소비자 결제             990원
토스 결제 수수료 3.5%  - 35원
Replicate 생성 비용    - 50원
─────────────────────────────
실제 가용 금액          905원

작가 70%                634원
플랫폼 순수익           271원
```

### 번들 상품
```
1회   990원  → 플랫폼 271원
5회  4,500원 → 플랫폼 4,500원 기준 동일 비율 적용
```

### 손익분기
```
1인 최소 생존: 월 200만원
필요 생성 수:  200만 ÷ 271원 = 7,380회/월 = 하루 246건
현실적 목표:   작가 3명 × 각 하루 100건
```

## 2. PMF 판단 기준

### 4주 안에 볼 숫자

| 지표 | 목표 | 의미 |
|---|---|---|
| 재사용률 (7일 내 재결제) | 30% 이상 | 제품이 진짜 쓸모 있다 |
| 작가당 일일 생성 건수 | 50건 이상 | 팬덤 활성화 |
| 첫 방문 → 결제 전환율 | 5% 이상 | 무료 → 유료 설득력 |

### 판단 룰
```
GO:    3개 모두 달성
PIVOT: 2주차에 하나도 안 됨 → 작가 교체 or 가격 조정
STOP:  2주차 일 10건 미만 → 즉시 중단
```

## 3. 화면 4개

### 화면 1: 작가 갤러리 (/)
- 작가 카드 2열 그리드
- 카드: 샘플 이미지 4장 + 작가명 + [만들기] 버튼
- 비로그인도 볼 수 있음

### 화면 2: 생성 스튜디오 (/studio)
- 보유 크레딧 있는 작가 화풍 선택
- 프롬프트 입력 + 예시 버튼
- 생성 → 결과 → 저장/재생성
- 크레딧 없으면 충전 유도

### 화면 3: 크레딧 충전 (/charge)
- 1회 990원
- 5회 4,500원 (10% 할인)
- 토스페이먼츠 결제

### 화면 4: 마이페이지 (/my)
- 카카오 소셜 로그인 정보
- 크레딧 잔액
- 생성 기록 썸네일

## 4. 작가 보호 장치

### 기술적 장치
- NSFW 강제 네거티브 프롬프트 (nsfw, nude, explicit, violence...)
- 작가 금지 키워드 등록 기능
- 일일 생성 한도 설정 (화풍 희소성 유지)
- 생성물 메타데이터 삽입 (작가명, 생성일, AI생성 명시)

### 작가 통제 권한
- 언제든 모델 비활성화 가능
- 생성 로그 열람 가능
- 금지 키워드 직접 설정

## 5. 작가 섭외 기준

### 1순위 타겟
- 이모티콘 출시 경험 있는 작가
- 오리지널 캐릭터 보유
- 팔로워 1,000~10,000명
- AI에 중립~긍정적 입장

### 섭외 DM (3줄)
```
안녕하세요, 작가님 캐릭터로
팬들이 직접 이미지 만드는 서비스 개발 중입니다.
수익 70% 드리고 언제든 내리실 수 있어요. 관심 있으세요?
```
