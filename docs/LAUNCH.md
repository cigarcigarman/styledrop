## 3. 런칭 후 4주 (PMF 검증 기간)

### 주차별 목표

**1주차: 데이터 수집**
```
목표: 첫 100건 생성
행동: 
- 작가 팬덤 반응 관찰
- 에러/불편 신고 즉시 대응
- 재방문 유저 파악
판단: 일 10건 미만 → 작가 교체 검토
```

**2주차: 재사용률 측정**
```
목표: 재사용률 15% 이상
행동:
- OPERATIONS.md 쿼리로 재사용률 측정
- 1회 쓰고 안 오는 이유 파악
  (생성 결과 만족도? 가격? 불편한 UX?)
판단: 5% 미만 → 무료 체험 횟수 조정 또는 가격 변경
```

**3주차: 전환율 측정**
```
목표: 방문 → 결제 전환율 5% 이상
행동:
- GA4 또는 Supabase Analytics로 퍼널 분석
- 갤러리 → 스튜디오 → 결제 이탈 지점 파악
판단: 2% 미만 → 무료 1회 체험 추가 고려
```

**4주차: 종합 판단**
```
GO    재사용 30% + 전환 5% + 일 50건 → 작가 추가
PIVOT 1-2개만 달성 → 약한 지표 원인 분석 후 수정
STOP  전부 미달 → 작가 교체 or 근본 피벗
```

## 4. PMF 이후 성장 경로

### 스케일 Phase 1: 작가 확대 (월 2-3명씩)
```
목표: 작가 10명
방법:
- 기존 작가가 다른 작가 소개
- 트위터 일러스트 커뮤니티 탐색
- 이모티콘 작가 협회/커뮤니티 접근
```

### 스케일 Phase 2: 수익 구조 추가
```
현재: 건당 990원
추가 옵션:
- 월 구독: 9,900원/월 무제한 (재구매율 30% 이상 확인 후)
- 고해상도: 1,990원/회 (2048px)
- 번들: 10회 8,900원 (11% 할인)
```

### 스케일 Phase 3: 바이럴 기능
```
- 생성 이미지 공유 기능
  → 워터마크: "@작가명 × StyleDrop"
  → 공유 시 트위터/인스타 최적화 사이즈 자동 변환

- 작가 랭킹
  → 이번 주 인기 화풍 노출
  → 신규 작가 배너

- 팬 컬렉션
  → 내가 만든 이미지 갤러리
  → 공개/비공개 설정
```

## 5. 리스크 시나리오 대응

### 시나리오 A: 작가 팬덤에서 AI 반발
```
대응:
1. 플랫폼 공식 계정은 침묵
2. 작가 본인이 대응 (작가가 원하는 방식으로)
3. "작가가 직접 만든 도구" 포지션 유지
4. 반발이 크면 해당 작가 일시 비활성화
   → 작가와 상황 논의 후 결정
```

### 시나리오 B: 경쟁사 출현
```
네이버/카카오가 유사 서비스 출시:
1. 작가 락인 강화 (정산 히스토리, 신뢰 관계)
2. 니치 집중 (한국 신진 일러스트 작가)
3. 작가 수익 비율 올리기 (70% → 75%)

스타트업이 유사 서비스 출시:
1. 작가 수 경쟁보다 품질 경쟁
2. 먼저 들어간 작가 커뮤니티 활용
```

### 시나리오 C: Replicate 비용 급등
```
대응:
1. RunPod 또는 자체 GPU 서버 고려
   (월 생성 1만건 이상 시 고려)
2. 가격 조정: 990원 → 1,200원
3. 생성 품질은 유지 (steps 줄이지 않음)
```
